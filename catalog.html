퍼시스 제품 카탈로그 앱 Ver.2

사무 가구 회사인 퍼시스의 제품 사진, 가격, 제품 특징 등의 내용으로 구성된 타블렛PC 용 어플리케이션. 기존의 종이 카탈로그에서 제공하지 못했던 다양한 콘텐츠, 빠른 제품 검색 기능, 가상 견적 기능을 담았고, 종이 카탈로그를 대체하는 것을 목표로 제작하였다.

Mobile - Android / iOS
화면설계, UI 디자인
2014

  • 직관적인 내비게이션

    빠른 제품 검색을 위해, 제품의 코드까지 외우고 있는 영업사원의 눈 높이에 맞춰 내비게이션을 구성하였다.

  • iOS & Android UI 가이드라인 준수

    iOS와 Android의 UI 가이드라인에서 권장하는 인터페이스를 채택해 각 OS 별 사용자에게 익숙한 경험을 제공하고자 하였다.

  • 콘텐츠 중심 UI 디자인

    가구는 큰 부피 때문에 고객이 구매 전에 실물을 보기 어렵다. UI 요소는 최소화하고 제품 사진을 화면에 꽉 차게 배치함으로써, 고객이 머리 속으로 제품의 이미지를 쉽게 떠올릴 수 있게 하였다.

  • 해상도 별 이미지 최적화

    이미지 자동 Resizing 기능을 적용해 각 사용자의 기기 해상도에 최적화된 제품 사진을 제공하는 한편, 콘텐츠 관리자의 업무 효율을 높이고자 하였다.
